Linux简介：

Linux是一种操作系统，它可以作为计算机硬件和软件之间的接口，使计算机程序和应用程序能够与计算机硬件进行交互。与其他操作系统如Windows和Mac OS X不同的是，Linux具有很高的可定制性和灵活性，使用户可以通过自定义和配置来获得更高的效率和更好的用户体验。

Linux由许多不同的组件组成，其中最核心的组件是内核，它负责管理计算机硬件资源并提供基本的系统服务。除了内核之外，Linux还有众多的系统工具和应用程序，它们一起构成了一个完整的操作系统，并提供了大量的功能和服务，包括文件管理、网络管理、安全性管理和软件开发等。

由于Linux是开放源代码的，因此用户可以自由地查看和修改Linux的源代码，以满足他们的特定需求。这种开放性使得Linux成为许多软件开发人员和计算机专业人士的首选操作系统。Linux非常适合运行服务器和大型计算机集群，这些系统需要高度的性能和安全性。

总的来说，Linux是一种免费且开放源代码的操作系统，提供了高度的可定制性和灵活性，适用于各种不同的计算机硬件和软件配置。它在许多领域得到了广泛的应用，包括服务器、嵌入式系统、移动设备和高性能计算等。

Linux命令格式

command [-options] [parameter]

说明：

  • command：命令名
  • [-options]：选项，可以来对命令进行控制，也可以省略
  • [parameter]：传给命令的参数，可以是零个、一个或者是多个

注意：

[]代表可选

命令名、选项、参数之间有空格进行分割

Linux常用命令

文件操作命令

ls

作用：显示指定目录下的内容

语法：ls [-al] [dir]

说明：

  • -a 显示所有文件及目录（.开头的隐藏文件也会列出）
  • -l 除文件名称外，同时将文件形态（d表示目录，-表示文件）、权限、拥有者、文件大小等信息详细列出

注意：

由于我们使用ls命令时经常需要加入-l选项，所以Linux为ls-l命令提供了一种简写方式，即ll

cd

作用：用于切换当前工作目录

语法：cd [dirName]

说明：

  • ~表示用户的homemulu
  • . 表示目前所在目录
  • .. 表示目前目录的上级目录

举例：

  • cd .. 切换当前目录的上级目录
  • cd ~ 切换用户的home目录
  • cd/usr/local 切换到/usr/local目录

cat

作用：用于显示文件内容

语法：cat [-n]fileName

说明：

  • -n ：由1开始对所输出的行数进行编号

举例：

  • cat/etc/profile 查看/etc目录下的profile文件内容

more

作用：以分页的形式显示文件内容

语法：more fileName

操作说明：

  • 回车键： 向下滚动一行
  • 空格键： 向下滚动一行
  • b 返回上一屏
  • q或者Ctrl+c 退出more

举例：

more/etc/profile 以分页的方式显示/etc目录下的profile文件内容

tail

作用：查看当前文件末尾的内容

语法：tail [-f] fileName

说明：

  • -f :动态读取文件末尾内容并显示，通常用于日志文件的内容输出

举例：

tail/etc/profile 显示/etc目录下的peofile文件末尾10行内容

tail -20/etc/profile 显示/etc目录下的peofile文件末尾20行内容

tail -f/itcast/my.log 动态读取/itcast目录下的my.log文件末尾内容并显示

mkdir

作用：创建目录

语法：mkdir [-p] dirName

说明：

  • -p：确保目录名称存在，不存在的就创建一个。通过此选项，可以实现多层目录结构同时创建。

举例：

  • mkdir anle 在当前目录下，建立一个名为anle的子目录
  • mkdir -p anle/test 在工作目录下的anle目录中建立一个子目录为test，如果anle目录不存在，则建立一个。

rmdir

作用：删除空目录

语法：rmdir [-p] dirName

说明：

  • -p:当前目录被删除后使父目录为空的话，则一并删除

举例：

  • rmdir anle 删除名为anle的目录
  • rmdir -p anle/test 删除anle目录中名为test的目录，若test目录删除后anle为空，则也会被删除
  • rmdir anle* 删除名称以anle开头的目录

rm

作用：删除文件或者目录

语法：rm [-rf] name

说明：

  • -r：将目录及目录中的文件（目录）逐一删除，即递归删除
  • -f：无需确认直接删除

举例：

  • rm -r anle/ 删除名为anle的目录和目录中的文件及目录，删除请需要确认
  • rm -rf anle/ 无需确认，直接删除anle目录中的所有文件
  • rm -f hello.txt 无需确认，直接删除hello.txt文件

拷贝移动命令

cp

作用：用于复制文件和目录

语法：cp [-r] sourcr dest

说明：

  • -r：如果复制的时目录需要使用此选项，此时将复制该目录下的所有子目录和文件

举例：

  • cp hello.txt anle/ 表示将hello.txt文件复制到anle目录中
  • cp hello.txt./ hi.txt 将hello.txt复制到当前目录，并改名为hi.txt
  • cp -r an/ ./anle/ 将an目录和该目录下的所有文件复制到anle目录下
    • cp -r an/* ./anle 将an目录下所有文件复制到anle目录下，不包括an目录

mv

作用：为文件或目录改名、或将文件或目录进行移动到其他位置

语法：mv source dest

举例：

  • mv hello.txt hi.txt 将hello.txt文件该为hi.txt
  • mv hi.txt anle/ 将hi.txt文件移动到anle目录下
  • mv hi.txt anle/hello.txt 将hi.txt文件移动到anle目录下在改名为hello.txt
  • mv an/ anle/ 如果anle目录存在，将an目录移动到anle目录中

打包压缩命令

tar

作用：对文件进行打包、解包、压缩、解压

语法：tar [-zcxvf] fileName [file]

包文件后缀为.tar表示只是完成了打包，并没有压缩

包文件后缀为.tar.gz表示打包的同时还进行了压缩

说明：

  • -z：z表示的是gzip，通过gzip命令处理文件，gzip可以对文件压缩或者解压
  • -c：c表示的是create，即创建新的包文件
  • -x：x表示的是extract，实现从包文件中还原文件
  • -v：v代表的是verbose，显示命令的执行过程
  • -f：f代表的file，用于指定包文件的名称

常用的打包压缩命令

打包：

  • tar -cvf hello.tar ./* 将当前目录下所有文件打包，打包后的文件名为hello.tar
  • tar -zcvf hello.tar.gz ./* 将当前目录下所有文件打包并压缩，打包后的文件名为hello.tar.gz

解包：

  • tar -xvf hello.tar 将hello.tar文件进行压缩解包，并将解包后的文件放在当前目录
  • tar -zxvf hello.tar.gz 将hello.tar.gz文件进行解压，并将压缩后的文件放在当前目录
  • tar -zxvf hello.tar.gz -C /usr/local 将hello.tae.gz文件进行解压，并将解压后的文件放到/usr/local 目录

文本编辑命令

vi/vim

作用：vi命名是Linux系统提供的一个文本编辑工具，可以对文件内容进行编辑，类似于windows中的记事本

语法：vi fileName

说明：

  1. vim是从vi发展来的一个功能更加强大的文本编辑工具，在编辑文件可以对文本进行着色，方便我们对文本进行编辑处理，所有实际中vim用的比较多

  2. 要使用vim命令需要我们自己进行安装，只需要输入以下命令

vim

作用：对文件内容进行编辑yum，vim其实就是一个文本编辑器

语法：vim fileName

说明：

  1. 在使用vim命令编辑文件时，如果指定的文件存则直接打开此文件，如果指定的文件不存在则新建一个文件。
  2. vim在进行文本编辑时共分为三种模式，分别为命令模式，插入模式和底行模式。这三种模式之间可以相互切换，我们在使用vim时一定要注意我们当前所处的是那种模式。

针对vim中的三种模式说明：

​ 命令模式

  • 命令模式下可以查看文件内容，移动光标（上下箭头、gg、G）
  • 通过vim命令打开文件后，默认的是命令模式
  • 另外两种模式需要首先进入命令模式，才能进行彼此

​ 插入模式

  • 插入模式下可以对文件内容进行编辑
  • 在命令模式下按[i、a、o]任意一个，可以进入插入模式。进入插入模式后，下方会出现[insert]字样
  • 在插入模式按Esc键，回到命令模式

​ 底行模式

  • 底行模式可以对文件内容进行查找、显示行数、退出等操作
  • 在命令模式下[:,/]任意一个，可以进行底行模式
  • 通过/方式进入底行模式后，可以对文件进行查找
  • 通过 : 进入底行模式后，可以输入wq（保存并退出）、q！（不保存退出）、set nu（显示行号）

软件安装

软件安装的方式：

  • 二进制发布包安装

    软件已经针对具体平台编译打包发布，只要解压，修改配置即可。

  • rpm安装

    软件已经按照redhat的包管理规范进行打包，使用rpm命令进行按照，不能自行解决库依赖问题

  • yum安装

    一种在线的软件安装方式，本质上还是rpm安装，自动下载安装包并安装，安装过程中自动解决库依赖问题

  • 源码编译安装

    软件以源码工程的形式发布，需要自己编译打包

防火墙操作：

  • 查看防火墙状态（systemctl status firewalld 、firewall-cmd –state）
  • 暂时关闭防火墙（systemctl stop firewalld）
  • 永久关闭防火墙（systemctl disable firewalld）
  • 开启防火墙（systemctl start firewalld）
  • 开发指定的端口（firewall-cmd –zone=public –add-port=8080/tcp –permanent）
  • 关闭指定端口（firewall-cmd –zone=public –remove-port=8080/tcp –permanent）
  • 立即生效（firewall-cmd –reload）
  • 查看开放的端口（firewall-cmd –zone=public –list-ports）

注意：

  1. systemctl 是管理linux中服务的命令，可以对服务进行启动、停止、重启、查看状态等操作
  2. firewall-cmd 是linux中专门用于控制防火墙的命令
  3. 为了保证系统安全，服务器防火墙不建议关闭
